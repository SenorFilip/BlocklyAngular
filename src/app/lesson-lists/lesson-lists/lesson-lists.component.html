Lists

Lists are structures that can contain any type of variable, and they can contain as many variables as you wish.
Creating a list takes a bit more typing than creating a string, but a list is more useful than a string because it can be manipulated..
Here's how you create and print a list:

>> shopping_list = ['carrots', 'potatoes', 'tomatoes', 'pasta', 'salt']
>> print(shopping_list)
['carrots', 'potatoes', 'tomatoes', 'pasta', 'salt']
We can also print the third item in the shopping_list (tomatoes) by entering its position in the list (called the index position)
inside square brackets ([]), like this:

>> print(shopping_list[2])
tomatoes
You've probably noticed that we used the number 2 instead of 3. That is because lists start at
index position 0, so the first item in a list is 0, the second is 1, and the third is 2.
You can also access items from the end of the list. Just be careful because the last index starts from -1.
That may not make a lot of sense to humans, but it does to computers.



MANIPULATION

Adding items

To add items to a list, we use the append function. A function is a chunk of code that tells Python to do something.
In this case, append adds an item to the end of a list.

>> shopping_list.append('onions')
>> print(shopping_list)
['carrots', 'potatoes', 'tomatoes', 'pasta', 'salt', 'onions']
Removing items

To remove items from a list, use the del command (short for delete).
For example, to remove the fifth item in the shopping list, salt, do this:

>> del shopping_list[4]
>> print(shopping_list)
['carrots', 'potatoes', 'tomatoes', 'pasta', 'onions']
Something more to know:
  • You can get a subset of a string. Etc. print( shopping_list[2:5] ) which prints item from index position 2 up to (but not including) index position 5
  • They can contain mixtures of data types. Like numbers, strings and other lists.
  • We can join lists by adding them, just like adding numbers, using a plus (+) sign
