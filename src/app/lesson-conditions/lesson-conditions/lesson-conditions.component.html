Conditions

So far, we haven't been able to do anything conditionally; we cannot tell Python "do this in some cases and do this other thing" in others.
But this is the sort of logic we use everyday when making simple decisions.
Programming languages like Python have tools to mimic those real-world choices.
Many of the decisions we make throughout the day – "it's cold, so put on a coat"; "you're running late, so text your friend";
"you're hungry, so eat something" – have an if-then structure. Just like you, Python can make if-then decisions.
IF statement
>> age = 13
>> if age > 20:
print('You are too old!')
An if statement is made up of the if keyword, followed by a condition and a colon (:), as in if age > 20:. The lines following the colon must be in a block,
and if the answer to the question is yes (or true, as we say in Python programming), the commands in the block will be run.


IF-THEN-ELSE statement
In addition to using if statements to do something when a condition is met (True), we can also use if statements to do something when a condition is not true.
For example, we might print one message to the screen if your age is 12 and another if it’s not 12 (False).
The trick here is to use an if-then-else statement, which essentially says “If something is true, then do this; or else, do that.”

>> print("Want to hear a joke?")
Want to hear a joke?
>> age = 15
>> if age >= 12:
print("Last time I got caught stealing a calendar I got 12 months.")
else:
print("Shh. It's a secret.")
Last time I got caught stealing a calendar I got 12 months.
Because we’ve set the age variable to 15, and the condition is asking whether age is equal to 15, you should see the first print message on the screen.
If we change the age variable to a number lower than 12 we should se the second print message.


ELIF statement
We can extend an if statement even further with elif (which is short for else-if). For example, we can check if a person’s age is 10,
11, or 12 (and so on) and have our program do something different based on the answer.
These statements are different from if-then-else statements in that there can be more than one elif in the same statement:

>> age = 12
>> if age == 10: # 1
print("What do you call an unhappy cranberry?")# 2
print("A blueberry!")
elif age == 11: # 3
print("What did the green grape say to the blue grape?")
print("Breathe! Breathe!")
elif age == 12: # 4
print("What did 0 say to 8?")# 5
print("Hi guys!")
print("Why wasn't 10 afraid of 7?")
print("Because rather than eating 9, 7 8 pi.")
else:
print("Huh?")
What did 0 say to 8? Hi guys!
In this example, the if statement on the second line checks to see if the value of the age variable is equal to 10 at 1.
The print statement that follows at 2 is run if age is equal to 10. However, since we’ve set age equal to 12,
the computer jumps to the next if statement at 3 and checks if the value of age is equal to 11. It isn’t,
so the computer jumps to the next if statement at 4 to see if age is equal to 12. It is, so this time, the computer executes the print command at 5.

Something more to know:
  • As you probably noticed we put multiple print functions under the IF statement. They all get called when the statement is true.
    That is called a block of code. Their indentation has the be the same or else we will get a SyntaxError
  • You can get a subset of a string. Etc. print( shopping_list[2:5] ) which prints item from index position 2 up to (but not including) index position 5
  • They can contain mixtures of data types. Like numbers, strings and other lists.
  • We can join lists by adding them, just like adding numbers, using a plus (+) sign
