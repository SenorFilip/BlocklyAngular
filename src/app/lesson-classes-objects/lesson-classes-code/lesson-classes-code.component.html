Classes and Objects Coding Assignment
Create a class called Vehicle with attributes kind (String), color(String) and value(float).
It should have a constructor and a function called getDescription which returns a String that looks like this:
#kind# has a #color# color and has a price of #value# euro

Create two vehicles called v1 and v2.
v1 should be a red car with a price of 45000 euro.
v2 should be a green van with a price of 50000 euro.
Print their getDescription return values at the end.
