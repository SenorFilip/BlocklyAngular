Loops
Loops or iterations are used for repeating sequences of code. They're wery usefull for many programming problems.
Python has a WHILE and a FOR loop. We're gonna focus on the FOR loop. The loop consist of three pieces:
- begin: n = 1
- end: n <= 6
- change: n += 1

Begin is the first value in the range. The default value is 0.
End is the last value which the current value can't pass.
Change is the amount by which we increment or decrement the current value.

The print() method writes it's content to the console.
