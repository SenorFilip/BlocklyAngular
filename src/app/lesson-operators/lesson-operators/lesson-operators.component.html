Operators

Python uses boolean logic to evaluate conditions. The boolean values True and False are returned when an expression is compared or evaluated.

>> x = 2
>> print(x == 2)
True
>> print(x == 3)
False
>> print(x < 3)
True
Notice that variable assignment is done using a single equals operator "=", whereas comparison between two variables is done using the double equals operator "==". The "not equals" operator is marked as "!=".

Boolean operators

Sometimes, you'll want to deal with more than one logical expression at a time.
You might like to see if one thing is true and another thing is true, one thing or another thing is true, or maybe see if both things are true.
Python has a few special words to help you do all that: and, or, and not.

And
and is a special keyword in Python that figures out whether, given two expressions, both are true. If so, Python will say True ; if not, it'll say False.
Here's a few examples:
>> True and True
True
>> 2 > 0 and 4949 > 48
True
>> True and False
False
>> 2 > 0 and 4949 < 48
False
>> False and True
False
>> 2 < 0 and 4949 > 48
False
Or
or is another special keyword that figures out whether, given two expressions, one or the other or both are true.
If so, Python will say True ; if not, it'll say False. Here's a few examples:
>> True or False
True
>> False or True
True
>> 3 > 0 or 0 < 3
True
>> 3 < 0 or 0 > 3
False
>> 3 > 0 or 0 > 3
True
>> 3 > 0 or 0 < 3
True
>> True or True
True
Not
not is Python's last keyword for combining booleans. When you put it in front of something, it negates, or reverses, that expression.
So things that were true become false, and things that were false become true. For example:
>> not True
False
>> not False
True
>> not True and True
False
>> not False and True
True
>> not (3 > 0)
False
>> not (3 < 0)
True
