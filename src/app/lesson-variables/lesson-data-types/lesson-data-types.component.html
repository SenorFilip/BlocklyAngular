Data Types

In Python, data can include numbers, strings (pieces of text), or Booleans (True or False).
Python supports two types of numbers - integers and floating point numbers. (It also supports complex numbers, which will not be explained in this tutorial).

NUMBERS

Integers (int)

Integers are positive or negative whole numbers. Values such as 10, 32, −6, 194689, and −5 are integers, but 3.14 and 6.025 are not.

x = 8
y = 96855286564375657641
z = -1234
temperature = -10
In the Python code snippet above, x, y, z, and temperature are int.

Floats (float)

Not all numbers are whole numbers. Decimal points are used to represent values that can’t be described with whole numbers.
For example, you might have half (0.5) of an orange. Numbers that use decimal points are called floating-point numbers, or floats.

Example:

x = 17.8
y = 1.9
z = -23.61
temperature = -2.45

STRING

Strings are defined either with a single quote or a double quotes.

myString = 'hello there'
myString1 = "hello there"
The difference between the two is that using double quotes makes it easy to include apostrophes (whereas these would terminate the string if using single quotes).
The same applies for the other way around. A different approach is to use backslash(\') to use an apostrophe in String encapsulated by apostrophes.

BOOLEAN

Boolean is a type that can be considered True or False. Think of it as Yes and No

isGrownUp = True
isOld = False
Those will be useful later e.g. to filter data.
